# Fitnessrezepte App

GitHub-Pages-fähige Web-App für virtuelle Fitnessrezepte.

## Funktionen
- Ein Rezept pro Ansicht
- Vor- und Zurückblättern
- Rezeptliste über den Button **Rezepte**
- Favoriten speichern
- Teilen-Funktion
- Eigene Rezeptbilder als lokale SVG-Dateien
- Branding mit integriertem Fit-Coaching-Originallogo

## Projektstruktur
- `index.html` – komplette App
- `images/` – Rezeptbilder
- `brand/logo-fit-coaching.png` – Originallogo
- `brand/logo.svg` – SVG-Verweis auf das Originallogo
- `icons/` – App-Icons
- `manifest.webmanifest` – PWA-Metadaten
- `service-worker.js` – Offline-Cache

## GitHub Pages
1. Alle Dateien in ein GitHub-Repository hochladen
2. In GitHub unter **Settings > Pages** die Branch wählen
3. Nach dem Deploy die `index.html` direkt öffnen
